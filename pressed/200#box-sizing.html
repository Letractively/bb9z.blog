跨浏览器的box-sizing写法

属性的写法

盒模型就不在此多说了，在怪异模式内容的宽度包括padding和border。

IE6之后若要采用该盒模型可以这样定义：


div {
	box-sizing: border-box;			// Opera，IE9
	-moz-box-sizing: border-box;		// Firefox，Seamonkey
	-webkit-box-sizing: border-box;	// Chrome，Safari
	-ms-box-sizing: border-box;		// IE7，IE8
}

注：从我实测的结果来看，IE7到IE9均有效。其他浏览器只要不是太旧就应该都是支持的。

IE6？

如果一个页面是符合标准的，IE6会使用标准盒模型，如果想让IE6进入怪异模式可以借助DTD的使用，或在doctype前加入额外的xml声明，例如：

<?xml version="1.0" encoding="utf-8"?>

其他

box-sizing的属性有 border-box、content-box、auto和inherit，默认应该不继承。Mozilla支持额外的padding-box属性。
